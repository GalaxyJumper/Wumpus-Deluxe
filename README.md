# The Divided Realms
Revamp of our submission for the 2025 Microsoft Hunt the Wumpus competition.
New features include:
- OpenGL rendering for 3x performance gain
- Support for Github Codespaces and web editing
- 2.5d top down view
- More intuitive and interactive map making tool
- Visuals revamp including better shadows, water, cliffs, and more
- More intuitive controls
- More to come--this is still an in-development project!
The old project can be found at https://github.com/199No/UW-Rejects
